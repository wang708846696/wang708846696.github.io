闭包

要想明白python中装饰器，则需要先知道python中的闭包规则

可以看出来函数也可以被引用，然后我们基于这个基础我们了解下闭包。

我们看到我们在函数内部声明了一个内部的函数，然后将这个函数的地址返回给外部，然后在外部调用内部声明的函数。这个例子就是闭包的使用。

由于闭包引用了外部函数的局部变量，则外部函数的局部变量没有及时释放，消耗内存

如果想要修改外部函数中的变量

  • python3中使用nonlocal关键字
  • python2中使用列表对其内容改变

  1. def w1(func): ==>将w1函数加载到内存

对只有这两句，尤其第二句大有文章

在谈装饰器:

装饰器有什么作用：

  1. 引入日志
  2. 函数执行时间统计
  3. 执行函数前预备处理
  4. 执行函数后清理功能
  5. 权限校验等场景
  6. 缓存
